Tag's aprendidas

Deve começar

Todo documento html deve começar com !DOCTYPE,seguida pela tag html, que abre e fecha o arquivo html.

A tag head

Na tag head, deve conter todos os dados que não aparecem na parte visual, providencia informações gerais (metadados) sobre o documento, incluindo seu título e links para scripts e folhas de estilos.

A tag body

A body, define o corpo do texto, onde fica as partes que são visiveis.

A tag p

Define o paragrafo.

A tag link

A tag link, serve de caminho entre o documento atual e fontes externas. Podendo ter as propriedades como:

  • _blank: abre a página em uma nova janela/aba;
  • _self: abre a página na mesma janela;
  • _parent: abre a página na mesma janela do link;
  • _top: cancela todos os demais frames e abre a nova página no mesmo navegador.

A tag a

A tag a, ou ancoragem define um hiperlink, normalmenteutilizadapara ligar uma páginaa outra, seu atributomais importante é o href, que indica o destino do link. Por padrão, os links aparecerão da seguinte forma em todos os navegadores:

  • Um link não visitado é sublinhado e azul
  • Um link visitado é sublinhado e roxo
  • Um link ativo é sublinhado e vermelho
